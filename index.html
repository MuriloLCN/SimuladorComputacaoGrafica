Como executar o simulador

  1. Acesse o simulador com o botão abaixo
  2. Escolha um arquivo 3D .obj de sua escolha (alguns exemplos já estão no repositório deste trabalho)
  3. Para uma melhor experiência, aperte o botão "Normalizar Objeto" para evitar objetos gigantes
  4. Escolha o ponto de vista e o plano de visualização
  5. Carregue os dados para o simulador
  6. Inicie a simulação
  7. Para controlar a simulação, utilize o mouse para arrastar o objeto, o scroll wheel para alterar o ponto de vista e mouse + espaço para rotacionar o objeto
